# OurSpaces
A WIP React Native project that hopes to be a Myspace/Discord hybrid.

## Goals
- ***User Customization*** - My interest in this project is to better understand the security issues that came with html/css customization with Myspace and things like the Discord Enhancement Project. I want to provide as much customization as possible without exposing the user to vulnerabilities. 
- ***Real-Time Communication*** - voice group channels, allow users to create groups and customize them, and manage their own micro-communities 
- ***Mix of Modern Social Features Mixed with Myspace-like Elements*** - comments/likes, bulliten board like activity feed, adding a song to your profile, DMs, media uploads
